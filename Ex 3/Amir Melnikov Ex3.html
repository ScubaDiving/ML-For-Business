Handle categorical feature¶

Some are already encoded as numbers, but two are still objects.¶

Results Analysis¶

Based on the plots and the MSE, I would choose the Random Forest Regression model. The scatter plot for the Random Forest Regression model shows a more linear relationship between the actual and predicted values, indicating that the model is performing better than the Linear Regression model. Additionally, the MSE for the Random Forest Regression model is lower than the MSE for the Linear Regression model on both the training and testing data, which further supports the choice of the Random Forest Regression model. It is worth mentioning that this is not neceseraly a good result as the diffrence between the RFR train and test MSE is very large meaning the model overfitted even though the test MSE is lower than the one in the LR.
